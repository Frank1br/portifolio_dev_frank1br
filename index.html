Mobile Development

Este projeto é um aplicativo desenvolvido em Flutter, com back-end em PHP e banco de dados gerenciado no MyPhpAdmin. O objetivo do app é facilitar a intermediação de vendas de produtos químicos para empresas, proporcionando uma plataforma segura e eficiente para conectar fornecedores e compradores do setor.
